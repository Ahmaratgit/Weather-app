天気アプリ 🌸

このアプリは、Node.js、Express.js、EJS、Tailwind CSSを使って作った天気アプリです。
日本風のデザインが好きなので、シンプルで見やすいUIを意識して作りました。このプロジェクトを通して、APIの使い方、レスポンシブデザイン、EJS、フロントエンドデザインなどを練習しました。

このアプリでは、

* 現在の気温 * 最高気温・最低気温 * 日の出・日の入り * 雨の情報 * UV指数 などを表示できます。

また、

* ガラスモーフィズムUI * アニメーション * 光るカーソルエフェクト * 日本語UI * モバイル対応 なども実装しました。

使用技術

* Node.js * Express.js * EJS * Tailwind CSS * Open-Meteo API
